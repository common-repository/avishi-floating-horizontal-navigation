Keep your primary Navigation at the top as you scroll down the page.

== Description ==

Wordpress Plugin to Keep your primary Navigation at the top as you scroll down the page. While page scroll navigation will stay sticky on top of the Browser window. 


Easy to use:

* Just Intall
* Specify the Primary Navigation ID (HTML attribute) on Plugin setting page.
* Thats it!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip Avishi Floating Horizontal Navigation plugin download
1. Upload the entire avishi_floating_horizontal_navigation folder to your wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to Settings>>Avishi PayPal Payment
1. Configure the Avishi Floating Navigation, i.e. Specify Primary Navigation Container ID (HTML attribute).
1. Click the "Save Settings" button
1. That's it!
